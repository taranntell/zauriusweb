What's New in Fallinorg

Fallinorg 1.0.0 is now officially launched! Built on customer feedback from our beta program.

Version 1.0.0

Official Launch

We're thrilled to announce the official launch of Fallinorg 1.0.0! This release brings together all the most-wanted features from our beta program, refined based on your valuable feedback.

Highlights

  • Advanced OCR Support: Extract and analyze text from images for intelligent classification.
  • Comprehensive File Format Support: Full English language support for Microsoft Office (Word, Excel, PowerPoint), PDF, EPUB, Markdown, Text, RTF, CSV, and many more formats.
  • Quick Look Thumbnails: Visual previews integrated directly in the app make file recognition faster and more intuitive.
  • Finder Integration: Right-click any file in Finder to instantly add it to Fallinorg for classification from anywhere on your Mac.
  • Privacy First: All classification happens completely offline on your device. Your sensitive data never leaves your Mac.
  • Apple Silicon Optimized: Built exclusively for Apple Silicon to deliver the best AI performance possible.
  • AI-Powered Understanding: Uses advanced Sentence Transformers to truly understand file content, not just keywords.
  • Custom Categories: Full control to create your own categories and define exactly where files should be moved after classification.

Supported File Formats

Documents: TXT, PDF, MD (Markdown), RTF, HTML, EPUB, DOC, DOCX

Spreadsheets: CSV, TSV, XLSX, XLSM, XLTX, XLTM

Presentations: PPTX, PPTM, POTX, POTM

Data & Config: JSON, XML, YAML, TOML, INI, CONF, LOG

Images: PNG, JPG, JPEG, HEIC, HEIF, TIFF, TIF, BMP, WEBP

Thank you to all our beta testers who helped shape Fallinorg into what it is today. This is just the beginning!

Beta 2.0

Pre-Sale Release

New Features

  • Instant classification on Apple Silicon: Now powered by ONNX Runtime, file classification runs almost instantly compared to the previous 1.2 seconds per file.
  • Seamless updates: Over-the-Air (OTA) update support added, so future versions can be installed directly in the app.
  • Bigger batch processing: Process up to 150 files at once, tripling the previous 50-file limit.

Enhancements

  • Smaller, faster, lighter: App size reduced by 73%, from 456 MB to just 125 MB. Downloads are quicker, updates smoother, and storage use is minimal.
  • Smarter text encoding support: Automatically detects and supports UTF-8, UTF-16, ASCII, Latin-1, Windows CP1252, and macOS Roman for better accuracy.
  • Cleaner PDF classification: Identifies meaningful content while ignoring noise such as Acknowledgements, Index pages, or About sections.
  • Faster startup with caching: Embeddings are now cached between app sessions, improving performance and reducing loading times.
  • Sharper language detection: Improved English detection and better handling of multi-language files to minimize irrelevant content.

Bug Fixes

  • More stable performance: Fixed multiple bugs and crashes, including file path handling issues, for a smoother and more reliable experience.
  • No file size limits: Large files are now fully supported, removing the old 10 MB restriction.
