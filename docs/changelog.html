What's New in Fallinorg

Fallinorg Beta 3.1.0 introduces new features, enhancements and fixes, detailed below.

Beta 3.1.0

Pre-Sale Release

New Features

  • EPUB Support: Added support for EPUB files, bringing total supported formats to 15.
  • TSV Support: Added support for Tab-Separated Values (TSV) files for better data file organization.
  • Finder Integration: Right-click files anywhere on your Mac and choose 'Organize with Fallinorg' from the context menu to instantly organize them.

Enhancements

  • Improved HTML Parsing: Better handling of large HTML files for more accurate classification and faster processing.

Bug Fixes

  • General Stability: Various small fixes and improvements for better reliability.

Beta 3.0

Pre-Sale Release

New Features

  • Custom Categories: Create your own categories and edit the built-in ones to fit your needs.
  • Category Templates: Quickly add ready-made category sets without starting from scratch.
  • Broader File Support: Classify and preview more formats, including Text, PDF, Markdown, Rich Text (RTF), HTML, LOG, CSV, JSON, XML, YAML, TOML, INI, and configuration files.
  • Quick Look Preview: View a file's content instantly without opening it in another app.
  • Top-5 Matches: See the five best category suggestions for each file, making manual re-sorting easier.

Enhancements

  • Faster Performance: Reduced delays when loading cached data and processing large batches.
  • Refined Interface: Updated settings page and small UI improvements across the app for a cleaner experience.

Bug Fixes

  • General Stability: Fixed various issues and minor slowdowns to keep classification reliable and smooth.

Beta 2.0

Pre-Sale Release

New Features

  • Instant classification on Apple Silicon: Now powered by ONNX Runtime, file classification runs almost instantly compared to the previous 1.2 seconds per file.
  • Seamless updates: Over-the-Air (OTA) update support added, so future versions can be installed directly in the app.
  • Bigger batch processing: Process up to 150 files at once, tripling the previous 50-file limit.

Enhancements

  • Smaller, faster, lighter: App size reduced by 73%, from 456 MB to just 125 MB. Downloads are quicker, updates smoother, and storage use is minimal.
  • Smarter text encoding support: Automatically detects and supports UTF-8, UTF-16, ASCII, Latin-1, Windows CP1252, and macOS Roman for better accuracy.
  • Cleaner PDF classification: Identifies meaningful content while ignoring noise such as Acknowledgements, Index pages, or About sections.
  • Faster startup with caching: Embeddings are now cached between app sessions, improving performance and reducing loading times.
  • Sharper language detection: Improved English detection and better handling of multi-language files to minimize irrelevant content.

Bug Fixes

  • More stable performance: Fixed multiple bugs and crashes, including file path handling issues, for a smoother and more reliable experience.
  • No file size limits: Large files are now fully supported, removing the old 10 MB restriction.

Beta 1.0

Pre-Sale Release

New Features

  • Initial release: Core AI-powered file organization functionality.
  • Local processing: All file analysis happens on your Mac, ensuring complete privacy.
  • Offline operation: Works without internet connection.
  • Text and PDF support: Organize .txt and .pdf files in English.
  • Custom folder selection: Choose where your files are organized.

Enhancements

  • Apple Silicon optimization: Built specifically for M-series Macs.
  • Batch processing: Process up to 50 files at once.
  • Smart categorization: AI understands file content, not just names.
